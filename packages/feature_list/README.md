## feature_list

### Usage

```bash
feature_list <--list-file|-l> [--version-file|-v]
```

#### Options

- `--list-file` or `-l` : **(Required)** The path to the file that contains features list
- `--version-file` or `-v` : The path to the file that contains version functions
