# str_exist

A bash script that checks whether the target string exists or not

## Index

- [str_exist()](#str_exist)

### str_exist()

#### Example

```bash
str_exist "apple" "/path/to/file"
```

#### Arguments

- **$1** (string): target string
- **$2** (string): file path

#### Exit codes

- **0**: If target string exists
- **1**: If target string doesn't exist
